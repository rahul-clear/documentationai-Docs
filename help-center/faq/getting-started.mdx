## Overview

Start organizing your docs by creating a project.

## Steps

1. Log in to your Clear dashboard.
2. Click **New Project** in the sidebar.
3. Enter a project name and description.
4. Click **Create**.

Your project appears in the project list. You can now start adding docs and pages!
