## Email functionality and API

1. The API can be used to email Tax invoices, Simplified Tax invoices and their associated credit/debit notes. A link to download the invoice copies or attachments can be included as part of the email.
2. The email API can be used to trigger email of E-Invoices and payment reminders for single or multiple documents.
3. Taxpayers can also use the email functionality available on the web portal to send E-Invoices to their buyers.
4. The email would be triggered from ClearTax mailer services by default.
5. The email server and domain can be modified to a taxpayer-hosted server on demand, but this requires one-time setup and configuration. Note: the taxpayer would be responsible for the maintenance of the email server.
6. The email contents are predefined and can be modified on demand.
7. The reply-to email and contacts can be configured as required.
