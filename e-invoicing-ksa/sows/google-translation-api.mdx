## Translation API

1. The API can be used to translate the contents of a document from English to Arabic for Tax invoices, Simplified Tax invoices and their associated credit/debit notes.
2. All relevant and applicable fields will be translated from English to Arabic if the Arabic value is not already provided.
3. Translations are done using a third party package and will only have an accuracy of around 95% or more, with commonly used words having higher accuracy.
4. Translated information will be sent to ZATCA and will also be available on print PDFs and downloadable reports.
