## Annual maintenance - Customer ERP/PoS integration

**Scope of services for Annual maintenance of ClearTax - Customer ERP/PoS integration:**

It includes:

1. Address any failure such as bugs and technical issues on the module deployed for e-invoicing integration with the ClearTax portal.
2. Ensure the integration development is up to date with respect to any new fields required as per ZATCA guidelines to push data to the ClearTax platform.
3. Address any queries and clarification with respect to the integration development.
4. Maintain the reports, forms and layout built as part of the integration development.

It excludes:

1. Additional features or enhancements which are not contained in the original scope of work for integration.
2. Any changes arising due to changes in ERP/Database systems, system upgrades, or data schema changes by the customer or third parties onboarded by the customer.
