## 1. API URLs to be used/whitelisted

For API requests, use the below information.

### 1.1 Sandbox Environment (QA, Testing)

| API Type | Hostname | IP | Port |
|----------|----------|----|------|
| API | https://api-sandbox.cleartax.com | 193.122.66.8, 84.8.110.211 | 443 |

Please whitelist both the IPs on sandbox if it is required in your network environment or ERP.

**Base URL**:

```
https://api-sandbox.cleartax.com/middle-east/ksa/einvoicing
```

<Callout kind="info">
Please note that the above URL is for API requests only. The corresponding frontend web application URL where you can log in and verify the data is [https://app-sandbox.cleartax.com/middle-east/ksa/einvoicing/](https://app-sandbox.cleartax.com/middle-east/ksa/einvoicing/)
</Callout>
