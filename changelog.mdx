## Upgrading to the Latest Version

Follow these steps to upgrade Clear smoothly.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export your current configuration and documents to avoid data loss.

    <Callout kind="tip">
      Use the export API endpoint: `https://api.example.com/v1/export`.
    </Callout>
  </Step>
  <Step title="Update Package" icon="download">
    <CodeGroup tabs="npm,yarn">
      ````bash
      npm install clear@latest
      ````

      ````bash
      yarn add clear@latest
      ````
    </CodeGroup>
  </Step>
  <Step title="Run Migrations" icon="settings">
    Execute database migrations if applicable.

    ````bash
    clear migrate
    ````
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart your Clear instance and verify functionality.
  </Step>
</Steps>
